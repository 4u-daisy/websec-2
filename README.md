# Lab 2

## Online Multiplayer Race 

The goal of the game is to collect the maximum number of points and not to die :)
### install
```
git clone *repo* 
for win: cd/d folder 
npm install express --save-exact 
npm instal nodemon 
npm install socket.io --save-exact 
```

### using
```
nodemon backend.js 
*or* 
node backend.js 
```
